Disperse

The Dispersed System Formalism (DSF) was developed by the French scientist HervÃ© This as a method to describe food systems based on their constituent phases.


The disperse tool uses pseudo-random number generators to create DSF formulas containing the following food phases:

  • gas (G)
  • oil (O)
  • solid (S)
  • water (W)

The phases are taken to be associated topologically by a set of operators. Here, the operators used are:

  • dispersed-in (/)
  • included-in (@)
  • superposed-on (Ïƒ)
  • bicontinuous-with (Ã—)
  • in-addition-to (+)

In the original DSF papers an algorithm written with the proprietary software Maple was presented to generate lists of valid DSF formulas. disperse is written in javascript and is designed to provide a more convenient method of procedurally generating food formulas in the browser (for free).

On the Ternary page formulas containing three phases can be generated. This introduces some ambiguity, as O+G/W can be interpreted as (O+G)/W or O+(G/W). The ternary formulas generated can be "disambiguated" by converting the default infix notation to postfix notation.


Thus, instead of writing O+G/W we can write OG+W/, which is read as "oil and gas both dispersed in water".
